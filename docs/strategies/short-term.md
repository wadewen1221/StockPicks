# 短线激进策略

> 适用: 1-5 天持仓，强技术面 + 资金流入信号

## 策略逻辑

```
短线买入 = KDJ 金叉(技术) AND 量比 > 2(资金) AND 换手率 > 5%(活跃)
```

## 评分维度

| 维度 | 权重 | 满分条件 |
| --- | --- | --- |
| KDJ 金叉信号 | 35% | K 值上穿 D 值，且均在 20-80 区间 |
| 量比放大 | 25% | 当日量 / 5 日均量 ≥ 2 |
| 换手率 | 20% | 5%-15% 最佳 |
| MACD 趋势 | 20% | DIF > DEA 且柱状图为正 |

## 入场条件 (AND)

1. ✅ KDJ 金叉 (K 上穿 D)
2. ✅ 量比 ≥ 2.0
3. ✅ 换手率 ≥ 5%
4. ✅ 收盘价 > 5 日均线
5. ✅ 非 ST / 非停牌 / 非次新股 (上市 > 60 天)

## 出场条件

- 单日跌幅 > 7% 触发止损
- 持仓 5 个交易日强制卖出
- KDJ 死叉提前止盈

## 代码入口

```python
from selector import StockSelector

selector = StockSelector()
picks = selector.run_short_term_strategy(top_n=20)

for p in picks[:5]:
    print(f"{p['code']} {p['name']} - 评分: {p['score']:.1f}")
```
