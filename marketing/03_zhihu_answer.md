## 正文

作为一个 20 年 A 股老韭菜 + 程序员,今年我把自己用的选股工具开源了。Stock Picks V2,从数据下载到选股到回测全自动。

今天来分享下怎么搭起来的。

### 为什么不用现成的?

市面上的量化平台都有这些问题:

- **同花顺/通达信**:闭源、收费、没法写自定义策略
- **聚宽/米筐/优矿**:好是好,但要上传策略,平台倒闭了策略就没了
- **QStock**:功能少,不开源

所以 2024 年开始自己撸,V1 跑了 1 年觉得架构太烂,2025 年重写 V2。

### 整体架构

```
Vue 3 前端 ←REST API→ Tornado 后端
                          ↓
                    本地 JSON 数据(5500 只 × 1500 日)
                    APScheduler 定时任务(7:00/17:30/18:30)
```

部署:Docker Compose 一键起,不需要装 Python/Node。

### 核心能力

1. **4 套选股策略**(满分筛选制)

| 策略 | 适用 | 入选分 |
|---|---|---|
| 中长线价值 | 1-3 个月持有 | ≥95 |
| 激进超短 | 1-3 天博弈 | ≥100 |
| 稳健超短 | 1-2 周 | ≥100 |
| RSRS 阻力支撑 | 中短线 | ≥60 |

每套策略包含 4-5 个子维度(趋势、动量、位置、量价、波动),加权打分。

2. **17 个技术指标**

MA / MACD / KDJ / RSI / CCI / RSRS / BOLL / 成交量 / 换手率 ...

RSRS 是我特别推荐的——阻力支撑相对强度,在大盘下跌时也能找出相对强势的股票。

3. **真实回测**

Backtrader 框架,接入 A 股真实佣金:

- 印花税 0.05%(卖出)
- 过户费 0.001%
- 规费 0.003%
- 佣金 0.01%

输出:SQN、年化、夏普、最大回撤。

4. **季度财务指标(7 项自动下载)**

- 营业总收入、归母净利润、扣非净利润
- 每股净资产、资产负债率
- 总股本、流通股本

数据源:baostock(主) + akshare(兜底)。2026 年 5 月 10 号 02:00 自动跑一次。

5. **定时任务全自动**

```python
07:00  早间资讯(腾讯财经热点)
17:30  历史数据增量下载
18:30  选股+回测
5/9/11 月 10 号 02:00  季度财报
```

跑完就推到 Web,我睡前打开看一眼。

### 怎么用?

不需要装 Python/Node,装个 Docker Desktop 就行:

```bash
git clone https://github.com/wadewen1221/StockPicks.git
cd StockPicks
docker-compose up -d

# 前端: http://localhost:8080
# 后端 API: http://localhost:5001/api/health
```

或者本地开发模式:

```bash
python backend/start_v2.py  # http://localhost:5001
```

### 适合谁用?

✅ 个人投资者,想自己跑量化策略
✅ 程序员,想拿现成框架学习
✅ 量化研究者,需要 A 股数据 + 回测引擎

### 不适合谁?

❌ 想要"一键躺赚"——没有这种东西
❌ 不想折腾 Docker/Python
❌ 需要实时行情(我只用日线)

### 风险提示

❗ **不构成任何投资建议**
❗ 历史回测业绩不代表未来收益
❗ 投资有风险,入市需谨慎
